# The Fates App

A mythology-based prediction game where real-world events are reframed as acts of the Greek gods.

## Files

- `index.html` — Landing page (main site)
- `profiles.html` — God character profiles
- `decrees.html` — Prediction card UI
- `season-one.html` — Season One story arcs
